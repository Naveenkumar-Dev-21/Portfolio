Cipher Hide

A steganography tool that allows users to hide text messages within **image, audio, and video files**, ensuring covert communication.

Technologies Used: Python, Pillow (PIL), Pydub (for audio/video), Bitwise operations.

Features:

  • Embeds hidden messages into images, audio, and video without visible alterations.
  • Extracts hidden messages from modified images, audio, and video files.
  • Supports various file formats (PNG, BMP for images; WAV, MP3 for audio; MP4, AVI for video).
  • Intuitive command-line interface for ease of use.
